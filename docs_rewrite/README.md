# Tothys Dungeon Tools TBC Rewrite Docs

Dieser Ordner ist der Arbeitsbereich fuer den Rewrite von `Kiesel Dungeon Tool`.

Wichtige Trennung:

- sichtbarer Addon-Name im Spiel: `Kiesel Dungeon Tool`
- interner Rewrite-Ordner und Dateinamen: `Tothys-Dungeon-Tools-TBC`

Diese Doku ist nur fuer den Rewrite gedacht, damit wir den Neubau nicht mit dem alten Addon vermischen.

Die wichtigsten Dateien hier:

- `PLAN.md`
- `DATA_MODEL.md`
- `NORMALIZED_CONTENT_MODEL.json`
- `../Tothys-Dungeon-Tools-TBC/Tothys-Database.lua`

Die Datei `NORMALIZED_CONTENT_MODEL.json` ist ein generierter Export aus der alten DB und dient als Arbeitsbasis fuer das neue Addon.
Die Datei `Tothys-Database.lua` ist die Runtime-Datenbank fuer den Rewrite.
