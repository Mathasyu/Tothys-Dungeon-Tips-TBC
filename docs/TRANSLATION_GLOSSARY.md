# Translation Glossary

This file defines preferred wording for German TBC tip translations.

Goal:

- keep tooltips and tip text readable for real WoW players
- allow common WoW community wording
- avoid inventing a different translation style in every instance

## General rule

- UI labels, config labels, and class/role checkboxes should stay clear and formal.
- Tip text may use player-natural WoW wording when it reads better.
- Keep sentences short and raid-friendly.

## Preferred terms

## Style guidance

- Prefer `Kick den Cast` over overly formal alternatives.
- Prefer `dispellen` over `reinigen` in generic dungeon or raid tips.
- Prefer `Tank`, `Heiler`, `DD` in prose.
- Prefer community wording over textbook German when both mean the same thing.
- Do not turn every sentence into slang; keep it readable for all players.
